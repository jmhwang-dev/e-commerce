# Purpose
- Health & Beauty 카테고리 핵심 제품의 매출 극대화를 위해 데이터 기반으로 성장 저해 요인을 식별하고 모니터링하는 분석 플랫폼 구축

# Result
- 현업 담당자를 위한 대시보드를 제공해 데이터 기반 의사결정을 지원
- 대시보드 주요 내용:
  1. 실시간 매출 트렌드 시각화
  2. 수익성 높은 상품군 식별을 위한 실시간 BCG 세그먼트 시각화
  3. Top 10 제품 세부 정보
